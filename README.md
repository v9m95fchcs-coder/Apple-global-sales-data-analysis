# Apple-global-sales-data-analysis
This project perform exploratory data analysis on apple global sales data using python.

Project Overview :
The dataset contains global sales records of Apple products. Using Python data analysis libraries, the project explores patterns in sales, identifies high-performing products, and evaluates how factors like price, discount, and region affect total revenue.

Tools :
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Key Analysis Performed :
Revenue analysis
Product category performance
Regional sales analysis
Monthly sales trends
Discount impact on revenue
Customer rating analysis
Correlation analysis between variables

Key Insights :
Certain Apple product categories contribute the highest revenue globally.
Sales distribution varies significantly across regions.
Discounts influence purchasing behavior and overall sales volume.
Monthly trends reveal demand patterns for Apple products.

Skills Demonstrated :
Exploratory Data Analysis (EDA)
Data Cleaning and Preprocessing
Data Visualization
Business Insight Generation
Python for Data Analysis
